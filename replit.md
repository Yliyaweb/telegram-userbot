# Telegram Userbot Monitor

Этот проект представляет собой Telegram userbot для мониторинга входящих личных сообщений и уведомления через отдельного бота.

## Переменные окружения
Для работы необходимы следующие переменные:
- `API_ID`: Получите на my.telegram.org
- `API_HASH`: Получите на my.telegram.org
- `BOT_TOKEN`: Токен вашего бота от @BotFather
- `MY_CHAT_ID`: Ваш ID, куда будут приходить уведомления

## Установка
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте секреты в Replit.
3. Запустите `python main.py`.

При первом запуске userbot может потребоваться ввод кода подтверждения в консоли.
